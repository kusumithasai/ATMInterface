# ATMInterface

# Features
- Transaction History
- Withdraw, Deposit, and Transfer
- Session ends with Quit option
- Password validation

# How to Run
1. compile: javac ATMInterface.java
2. Run: java ATMInterface
3. Enter default credentials (user / 1234)

# Technologies Used
- Java Console Application
- Scanner Class for input
- Classes and Objects
